Azeru Marketplace

Магазин Azeru Marketplace представляет собой каталог решений, сервисов, образов окружений разнообразной сложности, готовых для быстрого автоматического развёртывания на платформе Azure.

Основная идея Azure Marketplace - дать независимому разработчику канал продаж своих решений по всему миру, а клиентам Azure - лёгкий способ развернуть функциональные решения в собственной инфраструктуре.
